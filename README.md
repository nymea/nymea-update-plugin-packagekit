nymea-update-plugin-packagekit
================================

This repository contains the nymea platform update plugin that integrates
PackageKit based systems with nymea's update controller infrastructure.

It discovers repositories and packages via PackageKit and exposes them to the
nymea daemon so systems can advertise available updates and perform upgrades.
